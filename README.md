# Harsh Srivastava — DevOps & Frontend Portfolio

A modern, high-performance portfolio website showcasing projects in DevOps, cloud infrastructure, and modern frontend development.

## 🚀 Live Site
Check it out here: **[https://portfolio-website-dun-phi-97.vercel.app](https://portfolio-website-dun-phi-97.vercel.app)**

## 🛠️ Project Showcase
This portfolio highlights a balance between infrastructure and user experience:
- **DevOps Focus**: Features projects involving Docker (3-Tier apps), Nginx reverse proxies, AWS (VPC, S3, ECS, RDS), and CI/CD pipelines.
- **Frontend Excellence**: Includes high-fidelity UI clones (Apple website clone with 3D interactions) and AI-driven applications.
- **Interactive Experience**: Interactive skill graphs, masonry project layouts, and smooth animations.

## 💻 Tech Stack
- **Framework**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [Three.js](https://threejs.org/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/) + [Radix UI](https://www.radix-ui.com/)
- **Deployment**: [Vercel](https://vercel.com/) (Integrated with GitHub for CI/CD)

## 🏗️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/slayerhxrsh/portfolio-website.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Deployment
The project is configured for one-click deployment via Vercel. Any changes pushed to the `main` branch will automatically trigger a new production build.
